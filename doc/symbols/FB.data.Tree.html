Class Detail
FB.data.Tree(comparator)
The tree class represents an interval tree based on a red / black tree. An interval tree is the optimal data structore for events. Insert is O(lg n) and (as the tree is balanced) search for a node can be achieved in O(lg n) time as well. See Cormen, introduction to Algorithms, Interval Trees for more information.
Parameters:
{Object} comparator
The comperator delegate. The delgate gets called whenever a node is added and two nodes have to be compared. Returns 0 when the start time of the first event is less than the start time of the second event. 1 for the opposite case.
